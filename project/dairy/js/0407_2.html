횟수 기반 반복

for

for ( 초기값; 조건; 증가식 ) { 실행문 } : ( ) 안의 조건이 true인 경우, 실행문 실행

  • for ( let i=0; i<5; i++ ) { console.log (`${i}번째 반복입니다.`) }
  • i가 5보다 작다면, 실행문을 실행, 그리고 다시 +1 하고 i가 5보다 여전히 작다면
    실행문 실행, 그리고 다시 +1 하고 ...
  • 결과 : 0번째 반복입니다 / 1번째 반복입니다 / .. / 4번째 반복입니다.

for와 배열

  • const todos = [ '우유구매','메일확인','필라테스수업' ]
  • for( let i=0; i < todos.length; i++ ) { console.log (`오늘의 할일 : ${ todos[i] }` ) }
  • 결과 : 오늘의 할일 : 우유구매 / 오늘의 할일 : 메일확인 / 오늘의 할일 : 필라테스

for와 배열 거꾸로 출력

  • const todos = [ '우유구매','메일확인','필라테스수업' ]
  • for( let i=todos.length; i>=0; i-- ) { console.log (`오늘의 할일 : ${ todos[i] }` ) }
  • 결과 : 오늘의 할일 : 필라테스 / 오늘의 할일 : 메일확인 / 오늘의 할일 : 우유구매

for in : 객체 반복

for ( 반복변수 in 배열 ) { 실행문 }

  • const todos = [ 'pt수업','셰이크 먹기','점심식사']
  • for ( const i in todos ) { console.log ( `${i} 번째 할일 : ${todos[i]}` ) }
  • = 0번째 할일 : pt수업 / 1번째 할일 : 셰이크 먹기 ...
  • 배열의 인덱스 번호 만큼 반복 출력
  • 인덱스 번호를 출력하여 각각 i에 담음

for of : 배열 반복

for ( 반복변수 of 배열 ) { 실행문 }

  • const todos = [ 'pt수업','셰이크 먹기','점심식사']
  • for ( const i of todos ) { console.log ( `오늘의 할일 : ${todos[i]}` ) }
  • = 오늘의 할일 : pt수업 / 오늘의 할일 : 셰이크 먹기 ...
  • 배열의 인덱스 번호 만큼 반복 출력
  • es6에 새로 생겨 IE 실행 불가
