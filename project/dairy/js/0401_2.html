js 기본 용어

식별자

이름을 붙일 때 사용하는 단어

  • 키워드를 사용 안됨
  • 숫자로 시작 불가
  • 특수 문자 : _, $ 만 허용
  • 공백 문자 포함 불가
  • var a 식별자="짜장짱";

표현식

값을 만들어내는 간단한 코드

  • 예시 : const, 10, +, '짜장'

문장

표현식이 하나 이상 모인 것

  • 예시 : var varable="자고"+"싶다"
