제이쿼리 시작하기

제이쿼리

$(document).ready(function( ){ });

  • $ : 제이쿼리 시작할게 명령
  • $ ( ' html 요소' ) : 제이쿼리 객체 선언

아이디 선택하기

  • $ ( ' # html 요소' ).css ( 'color','red' )

클래스 선택하기

  • $ ( ' . html 요소' ).css ( 'color','red' )

자식 요소 선택하기

  • $ ( ' #abc > .def' ).css ( 'color','red' )

형제 요소 선택하기

  • $ ( ' #abc + .def' ).css ( 'color','red' )

속성으로 요소 선택하기

  • $ ( ' [title = this]' ).css ( 'color','red' )

first 필터셀렉터

첫번째 요소 선택 (first-child처럼)

  • var el = $( '#el > li : first' )
  • el.css ( 'color','red' );

last 필터셀렉터

마지막 요소 선택 (last-child처럼)

  • $( '#el > li : last' ).css ( 'color','red' );

even 필터셀렉터

짝수 index 선택 : 0, 2, 4, ... (홀수 선택처럼 보여짐)

  • $( '#el > li : even' ).css ( 'color','red' );

odd 필터셀렉터

홀수 index 선택 : 1, 3, 5, ... (짝수 선택처럼 보여짐)

  • $( '#el > li : odd' ).css ( 'color','red' );

eq 필터셀렉터

특정 index 선택 index 0부터 시작하는 것 잊지말기

  • $( '#el > li : eq ( n )' ).css ( 'color','red' );

gt 필터셀렉터

특정 index 보다 index가 큰 요소 모두 선택

  • $( '#el > li : gt ( n )' ).css ( 'color','red' );

lt 필터셀렉터

특정 index 보다 index가 작은 요소 모두 선택

  • $( '#el > li : lt ( n )' ).css ( 'color','red' );

has 필터셀렉터

특정 요소를 포함한 요소를 선택

  • $( '#el > li : has ( h2 )' ).css ( 'color','red' );
  • $( '#el > li : has ( p )' ).css ( 'color','red' );

not 필터셀렉터

특정 요소를 제외한 모든 요소를 선택

  • $( '#el > li : not ( .class )' ).css ( 'color','red' );
