자료형의 종류

자료형

  • 자료 : 프로그래밍에서 프로그램이 처리할 수 있는 모든 것
  • 자료형 : 자료의 형태에 따라 나눈 것
  • 종류 : 문자열 자료형, 숫자 자료형, 논리(불) 자료형

문자열 자료형

  • 따옴표 ('', "") 안에 작성
  • 문자열 중 따옴표를 작성할 경우, " '' ", ' "" ' 로 작성
  • (console) 특수문자 : \n 줄바꿈 \t 탭\\ 역슬래시

문자열 연산자

  • 기호를 사용해서 연산 처리
  • const a='짜장아\n'+'장난감'+'그만놀고'+'자자'
  • = 짜장아 (줄바꿈) 장난감그만놀고자자

문자열 선택

  • [ ] 사용하여 특정 문자열 선택
  • 0부터 카운트
  • ('짜장'[0]) = 짜

문자열 길이

  • .length 사용
  • 공백(빈 문자열)도 카운트
  • ('짜 장'.lenth) = 3

숫자 자료형

  • 따옴표 없이 작성
  • 소수점이 있는 숫자와 없는 숫자 모두 같은 자료형으로 인식

숫자(산술) 연산자

  • 기호를 사용해서 연산 처리
  • 종류 : + 더하기 - 빼기 * 곱하기 / 나누기 % 나눈 나머지

논리(불) 자료형

  • false 0 true 1

논리 연산자

  • a = b : 대입 연산자 (논리연산자 X) a에 b를 넣어!
  • a == b : a와 b가 같다 같으면 true 다르면 false
  • a !== b : a와 b가 다르다 다르면 true 같으면 false
  • a === b : a와 b가 자료형까지 같다 같으면 true 다르면 false

if 조건문

  • if(조건){실행문} 조건의 결과가 true면, if 실행문 실행
  • if(조건){실행문} else{실행문} if 조건에 맞지 않으면, 무조건 else 실행문 실행
  • if(조건){실행문} else if(조건문){실행문} if 조건에 맞지 않으면, 다음 if 조건 검증 후 실행

논리곱, 논리합

  • && : 논리곱 연산자 그리고
  • var x=(7>6) true && (1>8) false : false
  • || : 논리합 연산자 또는
  • var x=(7>6) true || (1>8) false : true

부정연산자

  • ! : not 연산자
  • !true = false
  • !false = true

템플릿 문자열

  • 템플릿 : console.log( ` 8000+100의 값은 ${q+k} 입니다. ` )
  • 기존 : console.log( ' 8000+100의 값은 ' + (q+k) + ' 입니다. ' )
  • 백틱 ``으로 작성됨
  • 자료형 구분 없이 한줄로 편리하게 작성할 수 있지만 익스플로러 지원 X
