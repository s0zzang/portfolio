function 1 function 2 바로가기

show, hide

요소를 보이고, 안보이게 하는 함수

  • 인자 : fast, slow, n
  • 보이기 : $ ( '.a' ) . show ( )
  • 숨기기 : $ ( '.a' ) . hide ( )
  • hide ( 1000, function ( ) { alert('숨김완료') } )
  • : 1초 동안 스르륵 없어진 후, alert 창 띄우기 콜백함수

width, height

요소의 너비와 높이를 변경하는 함수

  • 너비 : $ ( '.a' ) . width ( n )
  • 높이 : $ ( '.a' ) . height ( n )
  • a.height/width( ) : 높이
  • a.innerHeight( ) : 높이 + 패딩
  • a.outerHeight( ) : 높이 + 패딩 + 보더
  • a.outerHeight(true) : 높이 + 패딩 + 보더 + 마진

toggle

  • 인자 : fast, slow
  • $ ( '.a' ) . toggle ( 'slow' )
  • 토글이 적용된 버튼을 누르면 천천히 껐다 켰다
  • on( 'click' )과 함께 사용

fadeIn, fadeOut

서서히 보이고, 안보이게 하는 함수

  • 인자 : fast, slow, n
  • $ ( '.a' ) . fadeIn ( )
  • $ ( '.a' ) . fadeOut ( 'fast' )

fadeTo

보이고, 안보이고의 정도를 조절할 수 있는 함수

  • 인자 : 기간, 불투명도
  • $ ( '.a' ) . fadeTo ( 3000, 0.5 )

slideUp, slideDown, slideToggle

내려오고 올라오게 하는 함수

  • 인자 : fast, slow, n
  • 내려가기 : $ ( '.a' ) . slideDown ( )
  • 올라가기 : $ ( '.a' ) . slideUp ( 'fast' )
  • 토글적용 : $ ( '.a' ) . slideToggle ( 1000 )

animate

top, left, bottom, right 좌표값을 변경하는 함수
* hover와 함께 사용한 경우 : 움직인 횟수를 기억, stop( ).animate( ) 로 사용하기

  • 인자 : { top : n, left : n }
  • 인자 : { left : n1 }, n2 : n2 ms동안 n1만큼 이동
  • $ ( '.a' ) . animate ( { top:100, left:300 } )
  • $ ( '.a' ) . animate ( { top:100 }, 1000 ) . animate ( { left:200 }, 2000 ) animate 중복 적용 가능
  • delay( n )과 함께 사용 가능

text

텍스트를 변경하는 함수

  • 인자 : ' text '
  • $ ( '.a' ) . text ( '짜장이 일로와' )
  • $ ( '.a' ) . text ( ' ' ) : text 지우기
  • $ ( '.a' ) . text ( ) : text 담기

html

텍스트를 변경하는 함수, 태그 사용이 가능함

  • 인자 : ' text ', ' <tag> text </tag> '
  • $ ( '.a' ) . text ( '짜장이 일로와' )

empty

text를 지우는 함수 보더 등의 흔적은 남음, opacity, visibility 느낌

  • $ ( '.a' ) . empty ( '짜장이 일로와' )

remove

text를 지우는 함수 보더 등의 흔적 모두 삭제, display:none 느낌

  • $ ( '.a' ) . remove ( '짜장이 일로와' )

attr

속성을 변경할 수 있는 함수

  • 인자 : ( ' 가지고 올 속성 ', ' 변경할 값 ' )
  • $ ( '.a' ) . attr ( 'src', 'img2.png' )
  • $ ( 'a' ) . attr ( 'target', '_blank' )

val

form 요소의 value 값을 변경할 수 있는 함수

  • 인자 : ' 변경할 값 '
  • select 요소의 인자 : ' ex2 ' 순서 변경
  • select 요소의 인자 : [ ' ex2 ', ' ex3 ' ] active 상태로 출력
  • $ ( '.a' ) . val ( '이렇게 변경하겠습니다.' )
