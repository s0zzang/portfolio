함수의 선언

  • function ( 매개변수1, 매개변수2 ) { 실행문 }
  • ex ( 인자1, 인자2 ) function만 작성해선, 문서에 실행되지 않음

익명 함수

변수 식별자 = function ( ) { }

  • const fn = function ( ) { }
  • fn( ) 함수 2번 부르면 실행문 2번 실행
  • 화살표 함수로 연결되어 es6부터 많이 사용함

선언적 함수

function 함수이름 ( ) { }

  • function fn ( ) { }
  • fn( ) 위치 상관 없음
  • 함수의 호이스팅선언적 함수를 먼저 읽는 특징으로 function 위,아래 모두 호출 가능

return

함수의 최종적인 결과
function ( ) { ~ return } : { } 안의 값을 밖으로 배달하여 적용되게 함

  • function f ( x ) { return x * x }
  • const a = f( 3 인자)
  • console.log ( a )
  • return 있는 경우 : 9
  • return 없는 경우 : undefined

화살표 함수

변수 식별자 = (매개변수) => {실행문}
익명 함수에서만 사용 가능

매개변수가 없는 경우

  • const hi = ( ) => { return "안녕하세요?" }
  • const hi = ( ) => "안녕하세요?" return값만 갖는다면 { }, return 생략 가능

매개변수가 1개인 경우

  • const hi = ( user ) => { document.write ( user + "님, 안녕하세요?" )}
  • const hi = user => { document.write ( user + "님, 안녕하세요?" )} 매개 변수 괄호 생략 가능
  • hi ('짜장')

매개변수가 2개 이상인 경우

  • let sum = ( a,b ) => { return a+b }
  • let sum = ( a,b ) => a+b
  • const hi = user => { document.write ( user + "님, 안녕하세요?" )} return값만 갖는다면 { }, return 생략 가능
  • document.write ('두 수의 합 : '+sum (10,20))
