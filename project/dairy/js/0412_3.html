  • 인자 : class명 .찍으면 안됨

  • 삭제 : $ ( '.a' ) . removeClass ( box )
  • 토글 : $ ( '.a' ) . toggleClass ( box ) 클래스가 없으면 추가, 있으면 삭제

append, appendTo

형제 요소를 마지막 자식 요소로 포함시키는 함수
: 구조가 변경됨

  • A . append ( B ) : A 사이의 마지막에 B 추가
  • A . appendTo ( B ) : A가 B 사이 마지막에 추가
  • append : $ ( '.basket' ) . append ( '.apple' )
  • appendTo : $ ( '.apple' ) . appendTo ( '.basket' )
  • 담긴 순서 : basket - apple - 2 - 3

prepend, prependTo

형제 요소를 첫 자식 요소로 포함시키는 함수
: 구조가 변경됨

  • A . prepend ( B ) : A 사이의 처음에 B 추가
  • A . prependTo ( B ) : A가 B 사이 처음에 추가
  • prepend : $ ( '.basket' ) . append ( '.apple' )
  • prependTo : $ ( '.apple' ) . appendTo ( '.basket' )
  • 담긴 순서 : 3 - 2 - apple - basket
