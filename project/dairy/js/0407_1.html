배열

[요소, 요소, 요소, 요소]
여러개의 요소를 담기 때문에 식별자를 복수로 설정

생성

  1. 선언과 동시에 값 할당
  2. 선언 후 값 할당
  3. New Array 사용

1. 선언과 동시에 값 할당

  • const array = [1, 'a', true, function ( ) { }, { }, [1,5] ]
  • 자료형과 관계없이 하나의 그릇에 담을 수 있음

2. 선언 후 값 할당

  • let fruits = [ ]
  • fruits [ 0 ] = ' 사과 '
  • fruits [ 1 ] = ' 복숭아 '
  • fruits [ 2 ] = ' 참외 '
  • fruits [ 3 ] = ' 수박 '
  • 결과 : array (4) / 사과, 복숭아, 참외, 수박

3. New Array 사용

  • let animals=new Array()
  • animals [ 0 ] = ' 2 '
  • animals [ 1 ] = ' 3 '
  • 결과 : array (2) / 2, 3

배열 뒤에 요소 추가

push로 추가

  • const todos = [ '아침먹기','점심먹기' ]
  • todos.push( '저녁먹기' ) 추가
  • 결과 : array (3) / 아침먹기, 점심먹기, 저녁먹기

인덱스로 추가

  • const foods = [ '삼겹살','쫄순이' ]
  • foods[ 5 ] = ' 배떡 ' 인덱스 5에 추가
  • 결과 : array (6) / 삼겹살, 쫄순이, (2), (3), (4), 배떡

length로 추가

  • const foods = [ '삼겹살','쫄순이' ]
  • foods[ foods.length ] =2 = ' 배떡 ' 인덱스 2에 추가
  • 결과 : array (3) / 삼겹살, 쫄순이, 배떡

splice : 배열 요소 제거, 수정, 추가

제거

  • const items = [ '텀블러','아이패드','노트북' ]
  • items.splice(1,2) 인덱스 1부터 2개 삭제
  • 결과 : array (1) / 텀블러

수정

  • splice(수정할 index 번호, 1, '수정 text')
  • const items = [ '텀블러','아이패드','노트북' ]
  • items.splice(2,1,'갤럭시탭') 인덱스 2번째의 첫번째(노트북)를 '갤럭시탭'으로 수정
  • 결과 : array (3) / 텀블러, 아이패드, 갤럭시탭

추가

  • splice(추가할 index 번호, 0, '추가 text')
  • const items = [ '텀블러','아이패드','노트북' ]
  • items.splice(2,0,'갤럭시탭') 인덱스 2번째에 '갤럭시탭'을 추가
  • 결과 : array (4) / 텀블러, 아이패드, 갤럭시탭, 노트북
