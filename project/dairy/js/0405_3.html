자료형 변환

typeof() : 어떤 자료형인지 확인시켜주는 함수

confirm

*자료 - > 불 자료

  • true or false 로 자료형을 전환함
  • 예시 : input=confirm('수락하시겠습니까?') = 확인 true 취소 false

prompt

문자열 자료 입력

  • 사용자로부터 입력 받은 문자를 사용
  • prompt(' 사용자에게 보여질 문구 ',' 사용자가 입력할 문구 공백 ')

Number

*자료 - > 숫자 자료

  • Number(변수명) : 사용자가 입력한 변수명 문자/불 자료를 숫자 자료형으로 변환
  • NaN : Not a Number 숫자 + 문자인 경우

숫자연산자로 자료 변환

" number " - 0

  • typeof( "숫자" ) : string
  • typeof( "숫자" - 0 ) : Number

true * 1

  • typeof( true ) : boolean
  • typeof( true * 1 ) : Number = 1

true + 1

  • typeof( true ) : boolean
  • typeof( true + 1 ) : Number = 2
  • 불과 숫자를 + 연산자로 연결하면 숫자로 변환 + 합산됨
