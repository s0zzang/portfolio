es6

상수 키워드 : const

  • 상수명은 한 문서에 한 번만 사용 = css ID, 중복선언 불가
  • 상수명 선언시, 값 할당 필수 한 문서에 한 번만 선언할 수 있기 때문에
  • 할당된 값 수정 불가능 상수이기 때문에

변수 키워드 : let

  • 변수명은 한 문서에 한 번만 사용
  • 할당된 값 수정 가능 방법 : (let 없이) 변수명 : 값
  • 변수명 지정시, 값 할당 필수 한 문서에 한 번만 선언할 수 있기 때문에

변수에 적용이 가능한 연산자

복합대입연산자

  • += : 기존 변수값에 값 더하기
  • a += 1 => a = a+1
  • -= : 기존 변수값에 값 빼기
  • a -= 1 => a = a-1
  • *= : 기존 변수값에 값 곱하기
  • a *= 1 => a = a*1
  • /= : 기존 변수값에 값 나누기
  • a /= 1 => a = a/1
  • %= : 기존 변수값의 나머지 구하기
  • a %= 1 => a = a%1

증감연산자

  • 변수 ++ 후위 : 기본 변수 값 + 1 기존 변수값 리턴, 다음 턴에 + 1
  • 변수 -- 후위 : 기본 변수 값 - 1 기존 변수값 리턴, 다음 턴에 - 1
  • 변수 ++ 전위 : 1 + 기본 변수 값 + 1된 변수값 리턴
  • 변수 -- 전위 : 1 - 기본 변수 값 - 1된 변수값 리턴
