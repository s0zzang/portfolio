meta 태그

  • 목적 : 문서의 정보를 알려줌
  • 특징 : 단일태그

속성에 따른 분류

  • 한글 사용 : charset="UTF-8
  • 자동 이동 : http-equiv="refrest(새로고침)" content="(시간)(URL=이동경로)"
  • 검색엔진 키워드 설정 : name="keywords" content="키워드1, 키워드2"
  • 검색엔진 키워드 설명 : name="description" content="이소정의 포트폴리오입니다."
  • 작성자 : name="author" content="이소정"
  • 작성프로그램 : name="generator" content="notepad++"

property = og

open graph : SNS 공유시, 미리보기

  • og:title content="컨텐츠 제목"
  • og:description content="웹페이지 설명"
  • og:url content="웹페이지 url"
  • og:type content="홈페이지의 종류"블로그, 웹사이트 등
  • og:image content="썸네일"
  • og:site_name content="사이트 이름"

media query 관련

  • meta name="viewport" content="width=device-width, initial-scale=1.0"
  • - width = device-width : 너비는 장치의 너비와 동일하게
  • - initial-scale = 1.0 : 접속 초기화면 배율을 1로
