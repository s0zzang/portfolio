table

  • 구조 : thead, tbody, tfoot
  • 장점 : 데이터를 화면에 표시하기 효율적이고 직관적임
  • 단점 : 검색엔진(SEO)에 반영되지 않음
  • tr : table row (가로줄)
  • td : table data (한 칸의 데이터)
  • th : table headin (두꺼운 글씨, 가운데 정렬)
  • caption : table의 제목

table style

  1. head 태그에 style 태그 삽입
  2. head 태그에 link 태그로 css 문서 연결
  3. table 태그에 style, border 속성 삽입
  4. colgroup : 열 단위 스타일 지정을 위한 태그
    col style="" 스타일 변경
    col span="n" 선택할 열의 갯수

셀 병합

  • 행 병합 : rowspan="3" - 나를 포함한 행 3줄을 합칠게
  • 열 병합 : colspan="5" - 나를 포함한 데이터 5칸을 합칠게

웹접근성 : scope

  • 목적 : 시각장애인용 스크린 리더기에서 데이터를 읽어주는 방향 제시
  • 특징 : th 태그에만 사용 가능
  • 사용 : th scope="col" : 세로 방향으로 읽기
