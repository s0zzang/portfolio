video

속성

  • controls : 컨트롤바 (필수)
  • muted : 음소거
  • autoplay : 자동재생 (음소거 상태에서만 가능)
  • poster (="이미지 경로") : 썸네일 이미지
  • width, height : 너비, 높이

source

  • video 하위에서 경로를 적기 위한 단일 태그
  • src 속성 : 경로 작성

track

  • video 자막을 넣기 위한 단일 태그
  • src 속성 : 자막 파일(vtt) 경로 작성
  • kind 속성 : track의 종류
  • srclang 속성 : 언어 코드
  • labal 속성 : 자막 파일의 제목

주의점

  • 익스플로러8 이하 버전은 동영상 재생을 지원하지 않아서 추가 멘트 입력 필요
  • 서버에 올려야만 자막 확인 가능
  • 자동재생 시 썸네일 이미지 확인 어려움
