img

  • 목적 : html 파일에 이미지 넣기
  • 속성 : src(경로), alt(부연설명), title
  • 특징 : 단일태그

img에서만 쓰이는 alt의 3가지 용법

  1. 웹접근성 : 시각장애인용 스크린리더기에서 읽음
  2. SEO (검색엔진최적화) : 이미지 검색상위 노출
  3. 대체텍스트 : 이미지가 엑박 뜰 때 대신 출력

유의점

  • 주의 : 이미지 삽입시 html은 자동적으로 확장자를 px로 인식하기 때문에 단위 쓰지 말 것!
  • * css : 확장자를 인식할 수 없는 경우를 대비해 px를 꼭 써야함


figure, figcaption

  • figure : 본문 내용 안에 포함된 첨부자료를 삽입하기 위한 태그
  • figcaption : figure 요소의 제목

목적 : 웹접근성

  1. 첨부자료에 대한 설명을 컴퓨터에게 알려줘 검색 일치도(정확성)을 높임
  2. 삽화이미지 임을 컴퓨터에게 알려주기 위해

특징

  • img 태그 상위에만 사용됨
  • 사용시 자동 줄바꿈
  • figcaption : 1개의 figure 안에 1개의 캡션만 가능
  • figcaption : 이미지 위아래 상관업이 사용 가능
