path로 line 그리기

  • d = " M 시작x 시작 y L(line) 종료x 종료y "
  • M : 시작 / L : 직선 / H : 수평 / V : 수직 / Z : path 닫기
  • path d="M 300 200 L 500 100 H 50 V 300 z"

path로 곡선 그리기

  • d = " M 시작x 시작 y C x y(시작 고정점), x y(종료 고정점), x y(핸들) "
  • M : 시작 / L : 직선 / H : 수평 / V : 수직 / Z : path 닫기
  • path d="M 100 150 C 100 150, 300 50, 500 250"

선의 모양

  • stroke-linecap : 선이 끝나는 모양 ( butt 안쪽 사각 / round 둥근 / square 바깥쪽 사각 )
  • stroke-linejoin : 모서리 모양 ( bevel 각진 / round 둥근 )
