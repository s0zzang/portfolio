ifram

html 안에 다른 html 문서 삽입

속성
- name, src, frameborder, width, height
- name : 하이퍼링크 삽입을 위한 iframe 이름 지정
- frameborder : 테두리 있음 (1) 테두리 없음(0)
특징
- 삽입된 문서의 코드까지 기본 html 문서에 포함되기 때문에 검색엔진 최적화(SEO)에는 좋지 않음
- 하이퍼링크 연결시 a태그 target="iframe의 name" 유의
하이퍼링크 연결
- a태그 target 속성 = "iframe's name"

  • 군포시청
