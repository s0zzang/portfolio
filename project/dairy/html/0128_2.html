h

  • 목적 : 문서의 제목 표시
  • 사용 : h1 ~ h6
  • 숫자가 클수록 글자가 진하고 커진다.
  • h1은 검색엔진의 대주제로 인식되기 때문에 문서당 1개만 사용할 것
  • hgroup : h요소가 여러 개일 때 묶어주는 역할 (화면에 표시되지 않음)

p

  • 목적 : 문단 구성
  • p 요소 안에서 엔터를 입력해도 줄바꿈이 되지 않는다.
  • p 요소의 시작과 끝을 바로 알기 위해 p(첫번째 줄) 내용(두번째 줄) /p(세번째 줄)로 작성할 것

br

  • 목적 : 줄바꿈
  • 특징 : 단일태그

hr

horizontal row
  • 목적 : 수평선 삽입
  • 특징 : 단일태그

title

  • 목적 : 마우스를 올렸을 때 해당 단어의 제목을 표시
  • 특징 : 모든 태그에 사용이 가능한 속성
  • 예시 : 마우스 올려보기
