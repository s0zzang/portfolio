새로 추가된 텍스트 관련 요소

mark

  • 목적 : 형광펜 효과
  • 예시 : 와인은 누가 뭐래도 달달하고 끝 맛이 깔끔한게 최고다.

meter

  • 목적 : 퍼센트 시각화
  • 사용 : meter value="0.5"
  • 예시 : 이소정의 예상 취업율은 99%이다.

time

  • 목적 : 컴퓨터가 이해할 수 있는 형태로 날짜와 시간을 변경해줌 (화면에는 표시되지 않음)
  • 사용 : time datetime="1995-11-06" /time
  • 예시 : 내 생일이 다가오면 수능이 얼마 남지 않았어

ruby / rt

  • 목적 : 한자로 된 단어의 음 표시
  • 사용 : ruby[한자로 된 단어 rt(음)]
  • 예시 : 漢子한자

progress

  • 목적 : 진행 상황 표시
  • 사용 : progress value="(진행정도)" max="(최대치)"
  • 예시 : 절반 남음

details, summary

  • 목적 : 접고 펴는 기능
  • 특징 : summary와 ul/ol 함께 사용
  • 예시
summary 태그로 제목을 쓰지
  • 이렇게 details 태그와
  • summary, ul, li 태그를 통해
  • 내용을 접었다 폈다가 가능하지
