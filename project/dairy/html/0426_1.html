path

path

  • rect
  • x="10" y="20" width="200" height="100" fill="red"
  • x="50" y="170" rx="10" ry="10" width="100" height="100" fill="red" 모서리 둥근 사각형
  • circle
  • r : 반지름
  • cx="350" cy="250" r="30"
  • ellipse
  • rx : x축 반지름 / ry : y축 반지름
  • cx="200" cy="200" rx="100" ry="50" fill="yellow" stroke="green" stroke-width="20"
  • line
  • x1, y1 : 시작되는 꼭짓점 좌표 / x2,y2 : 끝나는 꼭짓점 좌표
  • x1="10" x2="400" y1="30" y2="300" stroke="blue"
  • polygon
  • 열린 path : points="0 0 , 200 100"
  • 닫힌 path : points="0 0 , 200 100 , 150 300"

path로 line 그리기

  • d = " M 시작x 시작 y L(line) 종료x 종료y "
  • M : 시작 / L : 직선 / H : 수평 / V : 수직 / Z : path 닫기
  • path d="M 300 200 L 500 100 H 50 V 300 z"

path로 곡선 그리기

  • d = " M 시작x 시작 y C x y(시작 고정점), x y(종료 고정점), x y(핸들) "
  • M : 시작 / L : 직선 / H : 수평 / V : 수직 / Z : path 닫기
  • path d="M 100 150 C 100 150, 300 50, 500 250"

선의 모양

  • stroke-linecap : 선이 끝나는 모양 ( butt 안쪽 사각 / round 둥근 / square 바깥쪽 사각 )
  • stroke-linejoin : 모서리 모양 ( bevel 각진 / round 둥근 )
