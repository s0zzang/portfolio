이미지맵을 이용한 하이퍼링크 연결

1개의 이미지에서 좌표를 입력해 여러 개의 하이퍼링크 연결

img

  • 속성 : usemap="#지도의이름"

map

  • 목적 : 지도의 이름 명명, area의 부모태그
  • 속성 : name="지도의이름"
  • 특징 : img의 usemap="#지도" = map의 name="지도"

area

  • 목적 : 지도의 상세값을 지정함
  • 속성
    • coords : 좌표값 (x1,y1,x2,y2)
    • shape : 모양 (사각:rect,원:)
    • href : 이동할 경로
    • alt : 설명값
  • 좌표값
    • (x1,y1,x2,y2)
    • x1, y1 : 시작점의 x값, y값
    • x2, y2 : 종료점의 x값, y값
    • 사각형 : 시작점(좌측상단 꼭짓점) 종료점(우측하단 꼭짓점)
