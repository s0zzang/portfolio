form

데이터를 입력하는 input의 부모태그

  • action 속성 : 서버 프로그램 URL
  • method 속성 : 입력된 데이터의 노출 방식
    • get : 노출 - 쇼핑, 공유해야되는 항목들
    • psot : 비공개 - 로그인 정보

input

type에 따라 특정 데이터를 입력/선택 할 수 있는 태그

type의 속성값

text : 문자

password : 비밀번호 - 입력시 *로 표시됨

email : 이메일 - @ 미입력시 제출 불가능

tel : 전화번호 - 아이폰 : 클릭시 숫자 키패드

date : 날짜

datetime-local : 일시

month : 연, 월

week : 주간

time : 시, 분

color : 색상선택

number : 숫자선택 - value를 통해 시작값 지정

search : 검색 - text와 차이점 : 지우기(X) 버튼 유무

url : url 입력 - '.com' 형식으로 필수 입력

range : 범위 - value(시작값) min(최소값) max(최대값) step(단위)

file : 파일 업로드

submit : 제출   reset : 초기화

radio : 단일선택 - label : text 눌러서 선택 가능

checkbox : 복수선택 - label : text 눌러서 선택 가능

추가 속성
  • name : 서버 참조값
  • value : 서버 전송값
  • autofocus : 웹접속시 자동으로 입력할 수 있는 상태
  • autocomplete : 자동완성 = "on / off"
  • checked : 웹접속시 선택된 상태 *radio, checkbox 속성에서 사용
  • disabled : 비활성화
  • readonly : 쓰기금지 *text 속성에서만 사용
  • required : 필수입력
  • title : text가 없는 경우 스크린 리더기를 위해 입력
  • placeholder : 클릭하면 없어지는 text 삽입
  • pattern : 특정 패턴(ex:전화번호, 주민번호)를 벗어나지 않도록 하는 속성
    • 방법 : pattern = " \d {자릿수} "
    • 예시 : pattern = " \d {6} - \d{7} "
