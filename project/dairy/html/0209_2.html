fieldset

form 태그의 자식 태그들을 한 그룹으로 묶는 태그

  • 사용 : input 아래에서 사용
  • 특징 : 포함된 태그들을 box로 묶음

legend

fieldset box의 title

  • 사용 : fieldset 아래에서 사용
  • 특징 : fieldset box 가운데에 text가 위치하도록 함

label

input의 이름을 붙여주는 태그

  • 사용 : 필요에 의해 input 위에서 사용
  • 특징 : radio, checkbox의 경우 label + input = text를 눌러도 선택 가능

select

option 태그와 함께 사용하여 select box를 삽입

option

  • 목적 : 보기 콘텐츠 입력
  • 사용 : <option value="콘텐츠"> 콘텐츠 </option>
