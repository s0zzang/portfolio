정의 목록 입력하기

dl

definition list 정의 목록

dt

description term : 정의할 단어

dl

define description : 단어에 대한 설명

특징

  • dd 사용시 자동으로 들여쓰기 됨
  • 한 개의 dl는 반드시 한 개 이상의 dt와 dd를 포함해야 한다.
  • 한 개의 dt 안에는 여러 개의 dd가 포함될 수 있다.

예시

짜장 (강아지, 6세)
짜장이는 2016년 3월에 태어난 갈색 푸들이다.
짜장이는 흥분하면 귀를 펄럭거린다.
짜장이는 턱을 만져주는 것을 아주 좋아한다.
우동 (강아지, 4살)
우동이는 홍익대학교 앞에 사는 강아지다.
똥개는 아니지만 그렇다고 어떤 종류인지 명확히 알 순 없다.
사람들이 정신없이 지나가도 평화롭게 풀밭에 앉아있다.
