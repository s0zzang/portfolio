링크 관련 태그

a

  • 목적 : 하이퍼링크 설정
  • 속성 : href(경로) / target(새로운 탭으로 열기, 현재 탭에서 열기 등)
  • href : mailto, tel, sms 함께 사용 가능
  • 예시 : 소짱홈
  • tabindex : 웹접근성 / tab을 통해 포커스하는 기능
    • tabindex="n" : 번호가 작을 수록 우선순위
    • tabindex="0" : tab으로 포커스 되지 않는 a 태그 외의 태그 선택 가능
    • tabindex="-1" : tab 포커스 해제

name anchor

  • 목적 : 해당 페이지 내 위치 이동
  • 사용
    • 이동을 희망하는 태그에 id로 이름 부여
    • a 태그 href="#이름"
    • 특징 : #으로 페이지 최상위로 이동
    • 예시 1 : nav로 이동하기
    • 예시 2 : 위로

하이퍼링크 이미지

  • 목적 : 이미지를 눌러 링크 이동
  • 사용 : <a <img/> /a>

하이퍼링크 기능을 가졌거나 함께 쓰는 다른 태그

  • 이미지맵
  • iframe

link

  • 목적 : 해당 html와 다른 문서(예:style.css)를 연결
  • 속성 : rel(관계:stylesheet) / href(경로)

nav

  • 목적 : 문서의 부분 중 현재페이지 내 또는 다른 페이지로의 링크
