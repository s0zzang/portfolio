텍스트 관련 요소

abbr

abbreviation : 약어

  • 목적 : 단어(축약어)에 마우스 오버시 해당 단어 설명
  • 예시 : EBS에 놀러갔더니 펭수를 만나 펭하라고 했다.
  • 사용법 : abbr title= "(펭수 하이!)" > (펭하) /abbr

address

  • 목적 : 주소, 연락처가 기울여진 글씨로 표시
  • 예시 :
    Address : Dong Soong Dong, Jong Ro Gu, Seoul
    Phone: 02-3673-0113

em

  • 목적 : 기울여진 글씨
  • 예시 : Address : Dong Soong Dong, Jong Ro Gu, Seoul
    Phone: 02-3673-0113

     * address와 em을 목적에 따라 적절히 사용해야 함

strong

  • 목적 : 강조
  • 예시 : 센스있는 웹디자이너 이소정입니다.

b

  • 목적 : 글씨를 굵게
  • 예시 : 센스있는 웹디자이너 이소정입니다.

     * 화면에는 똑같이 표시되지만 시각장애인을 위한 스크린 리더기에서 strong 태그는 크게 읽어주고
       b태그는 다른 텍스트와 동일하게 읽어줌

sub

  • 목적 : 첨자(작은 하단 글씨) 표시
  • 예시 : 배스킨 라빈스baskin robbins에서 내가 제일 좋아하는 맛은 아몬드 봉봉almond bongbong이다.

blockquote

  • 목적 : 인용구 표시
  • 특징 : 들여쓰기 + 줄바꿈 / 인용구 출처 밝히기 (화면에 표시되지 않음)
  • 예시 : 이소정은
    오늘 날씨 왕 춥다!
    라고 말했다.

  • 사용 : blockquote cite="(http://..)"> 오늘 날씨 왕 춥다! 라고 말했다.

ins / del

  • 목적 : 삭제글 및 추가글
  • 예시 :

    과제 29일까지 제출
    변경 : 과제 31일까지 제출!
