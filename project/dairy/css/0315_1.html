float position 바로가기

float

  • 배열하고 싶은 모든 요소에 float 속성 적용
  • 사용시, 누적순서가 한 층 올라가 사용하지 않은 요소들과 겹쳐짐
  • float : right / left

★ float 해제 방법 (줄바꿈) ★

  • 줄바꿈 할 요소 float:left + 부모 요소 너비 설정
  • 줄바꿈 할 요소 clear:left
  • 나중에 배움

★ 특징 ★

  • inline 요소에 float 사용시, inline을 block 처럼 바꿔줌

clear

float가 적용된 값을 해제하고 줄을 바꿈

  • 해제 / 줄을 바꾸고 싶은 요소에 clear 속성 적용
  • float : left - clear : left / both
  • float : right - clear : right / both

float 요소 대응 방법

1. 부모 : none
- 단점 : 자식이 부모 box에 포함 X
2. 부모 : float float:left or right
- 자식이 부모 box에 포함
- 단점 : 너비가 자동 설정되어 좁아짐
3. 부모 : ★ overflow ★ overflow:hidden
- 자식이 부모 box에 포함
- 너비 100%
- 단점 : 컨텐츠가 많아져 박스가 더 생기면 컨텐츠가 숨겨질 수 있음
4. 부모 : none + 새로운 자식 생성 div:clear:both; height:0; (overflow:hidden;)
- 자식이 부모 box에 포함
- text 적힌 경우 overflow:hidden 추가해 보이지 않게 함
- 단점 : 번거로움
5. 부모 : ★ 가상선택자 (after) ★ 부모:after{content:'', display:block; clear:both;}
- 자식이 부모 box에 포함
- after{content:'', display:block; clear:both;}
