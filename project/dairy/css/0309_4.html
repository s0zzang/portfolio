상대 단위

절대 단위 : px

em

  • 대문자 'M' 너비 기준으로 한 단위
  • 기준 : 부모요소
  • 1em : 약 16px

rem

  • 대문자 'M' 너비 기준으로 한 단위
  • 기준 : 최상위(루트) 요소

%

  • 기준 : 자신을 감싼 부모

vh

  • viewport height
  • 기준 : 브라우저 높이
  • 100vh : 브라우저 높이 100%
  • 50vh : 브라우저 높이 50%

vw

  • viewport width
  • 기준 : 브라우저 너비
  • 100vw : 브라우저 너비 100%
  • 50vw : 브라우저 너비 50%

vmin

  • viewport minimum
  • 기준 : 웹브라우저 너비와 높이 중 더 작은 값
  • 예시 : 768x900 5vmin = 768x5/100 = 38.4px

vmax

  • viewport maximum
  • 기준 : 웹브라우저 너비와 높이 중 더 큰 값
  • 예시 : 768x900 5vmin = 900x5/100 = 45px
