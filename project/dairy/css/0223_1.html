font와 관련된 속성

font

font : style - variant - weight - size - line-height - family

  • 여러 속성을 한 번에 쓸 수 있는 축약속성
  • 띄어쓰기로 구분
  • 선언순서를 지키지 않으면 NO 실행 - 필요없다면 생략

font-family

font-family : 폰트1, 폰트2, 폰트3

  • 폰트1 불가한 경우 폰트2, 폰트2도 불가하면 폰트3 적용
  • 띄어쓰기, '-' 등 특수문자가 사용되면 "(예:맑은 고딕)" 안에 입력

font-size

font-size : 10px / 1.5em

  • px : 13px 싸이즈
  • em : 1.3em 싸이즈

font-weight

font-weight : normal / bold

  • normal : 무겁다 무거워
  • bold : 무겁다 무거워
  • 단위 : 200 300 500 700 등 서체가 굵기를 가지고 있는 경우만 적용됨

font-style

font-style : 종류

  • normal : friday
  • italic : friday

font-variant

소문자 크기의 대문자로 변환

  • normal : The party is on Friday evening
  • small-caps : The party is on Friday evening

webfont

link 방식 (CDN)

  1. 구글폰트, 눈누에서 link 복사
  2. [html] head > link로 붙여넣기
  3. [css] font-family 입력하기

@import 방식

  • 구글폰트, 눈누에서 @import 복사
  • [css] @import, font-family 붙여넣기
  • [html] link로 css 연결하기
