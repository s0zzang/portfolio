content

내용 추가

  • after, before 가상 선택자와 함께 사용
  • tag : before { content : "짜잔~" }
  • tag : after { content : "끝!" }
  • content : open-quote (")

이렇게 씁니다.

counter-increment

일반 태그에 일련번호 붙이기

  • 방법 : 1) 일련번호를 붙일 태그에 counter-increment:"name"
  •            2) 일련번호를 붙일 곳에 content:counter(name, 속성)
  • tag { counter-increment : name }
  • tag:before { content-counter : ( name, 속성 roman, alpha, latin 등 ) }
