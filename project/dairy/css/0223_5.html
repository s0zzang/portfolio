content 관련

quotes

인용부호 설정

  • "여는부호( ▶ )" "닫는부호( ™ )"
이렇게 씁니다.

content

내용 추가

  • after, before 가상 선택자와 함께 사용
  • tag : before { content : "짜잔~" }
  • tag : after { content : "끝!" }
  • content : open-quote (")

이렇게 씁니다.

transform

  • transform : rotate : "회전각"deg
  • transform-origin : 회전축 / x축, y축
  • - 0%, 0% : 좌측상단
  • - 50%, 50% : 중앙
  • - 100%, 100% : 우측하단

counter-increment

일반 태그에 일련번호 붙이기

  • 방법 : 1) 일련번호를 붙일 태그에 counter-increment:"name"
  •            2) 일련번호를 붙일 위치에 content:counter(name, 속성)
  • 붙일 태그 : tag { counter-increment : name }
  • 붙일 위치 : tag:before { content-counter : ( name, 속성 roman, alpha, latin 등 ) }
