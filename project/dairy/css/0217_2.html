outline

속성
- width, color
- style : border와 동일
- offset : border와 얼만큼 떨어질지 수치 입력

outline offset = 10px

border 효과

radius : 둥근 모서리

사용방법
border - radius : 수치 px
적용순서
- 4개 : top left - top right - bottom right - bottom left
- 3개 : top left - top right, bottom left - bottom right
- 2개 : top left, bottom right - top right, bottom left 대각선 적용

border-radius:0 30px

collapse : 단일 테두리 설정 (table)

사용방법
border - collapse : collapse

border tip

사용방법
top 방향 tip : bottom color right, left transparent
bottom 방향 tip : top color right, left transparent
right 방향 tip : left color top, bottom transparent
left 방향 tip : right color top, bottom transparent
요소에 설정하는 경우, position:absolute 설정
