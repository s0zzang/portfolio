flex

- 부모에 display:flex 적용시 자동으로 float:left 처리됨
- clear 시킬필요 없음, '%' 단위 사용시 반응형 완성!

flex-direction

flex item 배치

  • flex-direction : row 수평 좌측 정렬 (1-2-3)
  • flex-direction : row-reverse 수평 우측 정렬 (3-2-1)
  • flex-direction : column 수직 좌측 정렬 (1-2-3)
  • flex-direction : column-reverse 수직 우측 정렬 (3-2-1)

flex-direction

flex : 자식이 부모보다 커도 얌전히 부모 안에 갇혀있음 (기본값)

  • flex-wrap : nowrap 줄바꿈 X 기본값
  • flex-wrap : wrap 줄바꿈 O 크기가 크다면 자동으로
  • flex-wrap : wrap-reverse 줄바꿈 O - 반대로 크기가 크다면 자동으로

flex 축약속성

예시 : flex-flow:row wrap;

  • flex-flow : row flex-derection wrap flex-wrap

교차축 방향 아이템 개별 배치

  • align-self : flex-start 교차축 시작점기준 아이템 개별 배치
  • align-self : flex-end 교차축 끝점기준 아이템 개별 배치
  • align-self : center 교차축 중앙으로 아이템 개별 배치
  • align-self : auto flex-container(부모)에 적용된 align-item값 상속
  • align-self : baseline 교차측 시작점에서 제일 먼 text 기준 개별 배치

플렉스 아이템 박스 크기 조절

사용법 : flex:[flex-grow] [flex-shrink] [flex-basis]

  • flex-grow : 부모보다 작을 때 모자란 곳을 늘리기
  • flex-shrink : 부모보다 클 때 넘치는 곳을 줄이기
  • flex-basis : 앞에 적은 비율대로 할지 말지 선언 0 : Do | 1 : Don't

주축 방향 아이템 배치

  • justify-content : flex-start 기본값
  • justify-content : flex-end 주축의 끝점 배치
  • justify-content : center 주축의 중앙 배치
  • justify-content : space-between 요소 간 여백 (양끝 배치 + 중앙정렬)
  • justify-content : space-around 박스마다 좌우 동일여백 10 box1 20(box1의 10 + box2의 10) box2 10
  • justify-content : space-evenly 모두 동일 여백 10 box1 10 box2 10

교차축 방향 아이템 배치

  • align-items : stretch 기본값
  • align-items : flex-start 교차축의 시작점 배치
  • align-items : flex-end 교차축의 끝점 배치
  • align-items : center 교차축의 중앙 배치
  • align-items : baseline 교차축에서 가장 멀리 떨어진 text 기준 배치
