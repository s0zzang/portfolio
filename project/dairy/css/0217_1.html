background

이미지 배경

속성
- image : url (경로)
- repeat : repeat-x / repeat-y / no-repeat 반복될 방향 설정
- position : 수직 top/bottom/center 수평 left/right/center
- attachment : fixed 배경이 스크롤되지 않고 고정됨
- size : 숫자(w,h) / cover 요소의 너비에 맞춰 / contain 요소의 높이에 맞춰
- origin : border-box (보더 왼쪽 위) / padding-box (패딩 왼쪽 위)
                content-box (내용 왼쪽 위) 이미지가 채워지기 시작하는 위치
주의점
- 속성과 축약 중복 사용시, 다른 속성으로 인식하여 각각 적용됨
   1. li(first) : background:gradient 2. li : background-color:#fff
- 다중 이미지 사용시, 콤마로 구분
- 배경이 표시될 공간(너비, 높이)가 있어야 배경이 표시됨

gradient

사용방법
- background : 그라디언트종류 (각도, 시작색상, 종료색상)
- 노트패드 즐겨찾기 통해 그라디언트 쉽게 적용 가능
- 낮은 IE에서 적용이 안되기 때문에 단색 배경화면과 함께 입력하기

linear-gradient(45deg, #ffefbf, #ffbfbf)
