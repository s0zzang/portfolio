선택자 2 선택자 1 바로가기

가상 클래스 선택자

: 앞으로 벌어질 상황을 선택하여 스타일 지정

  • 방법 : a : 종류 {style}
  • 종류 : link (방문 안했던 링크) visited (방문했던 링크)
        hover (마우스 오버했을 때) active (마우스 클릭했을 때)

가상 클래스 선택자 : 동적

: 양식의 입력칸 등에 클릭이나 탭을 했을 때

  • 방법 : input : focus {style}
  • 종류 : 로그인 ID 클릭(tab) 했을 때 테두리, 배경 지정

가상 클래스 선택자 : 내용 추가

- 추가된 내용은 화면 드래그 시 선택되지 않음
- 기본 input 요소 아이콘이 아닌 내가 원하는 아이콘 넣을 때,
    line 등에 사용 (이미지 넣는 방법 : 3월에 배움)

  • 앞 : tag:before { content : '내용' }
  • 뒤 : tag:after { content : '내용' }

전체 선택자 : *

- 브라우저별 기본값이 다르기 때문에 0으로 세팅 (초기화) 하고 시작할 때 주로 사용
- 선택자 우선순위에 따라 body(2점)보다 점수가 낮음

그룹 선택자

: 같은 스타일을 적용할 선택자들을 동시에 선택

  • 방법 : tag1, tag2, tag3 { style }

n번째 요소 선택

: 여러번 쓰인 태그의 n번째 요소만 선택하기

  • 방법 1 : tag : first-child
  • 방법 2 : tag : nth-child (n)
  • 종류 : (even) 짝수 (2n) 2번째 요소마다 - 짝수 (3n) 3번째 요소마다 - 홀수

첫번째 글자/줄 선택

  • 글자 : tag:first-letter
  • 줄 : tag:first-line
