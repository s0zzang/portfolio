웹용 벡터 이미지

svg 파일 불러오기

  • svg 파일 통채로 복붙
  • svg 파일이 길 경우 문제가 됨
  • : rect width="100%" height="100%" fill="#색상"
  • : circle cx="150" cy="100" r="80" fill="#색상"
  • : text x="150" y="125" font-size="60" text-anchor="middle" fill="#색상
  • img로 불러오기
  • : img src="images/face2.svg"
  • inline로 불러오기
  • svg 파일의 < svg > 태그 전체 복사 붙여넣기
  • object embed
  • : object - embed src="loader.svg - /object
  • data : 경로 / type : image/svg+xml
  • < object > data="images/face2.svg" type="image/svg+xml"> < /object >
  • object size : viewBox="0 0 500 500"
  • object data (svg) & img (png)
  • svg 지원 안되는 브라우저를 위한 크로스 브라우징
  • 1) object - data="loader.svg" type="image/svg+xml
  • 2) img - src="loader.png"
  • 3) p - svg 이미지가 지원되지 않는 브라우저입니다.

주의점

  • 미디어쿼리 및 호버 등의 동작은 svg 파일에서 적용해야 함
  • 일러스트에서 변경한 이름대로 svg 파일로 불러와짐
