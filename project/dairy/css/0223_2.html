text / 문단과 관련된 속성

text-align

- left 좌측 정렬 right 우측 정렬 center 중앙 정렬 justify 좌우 정렬
- 블럭요소만 적용됨

  • img 요소에 적용하기 : 1) 부모블럭에게 text-align 적용
  • img 요소에 적용하기 : 2) img에게 + display:block + margin:auto
  • 인라인 요소에 적용하기 : + display:block + margin:auto + text-align:center

line-height : 행간

- line-height 1.5em 적용 : 유산균 먹기

letter-spacing : 자간

- letter-spacing 1.2em 적용 : 유산균먹기

- letter-spacing -2px 적용 : 유산균먹기

word-spacing : 단어간

- word-spacing 1.5em 적용 : 유산균 먹기

direction : 글의 방향

  • rtl : 오른쪽 정렬 right to left
  • ltr : 왼쪽 정렬 left to right

white-space

화면 크기/box 크기에 따라 자동으로 줄바꿈되는 속성을 제어함

  • wrap : 줄바꿈 O
  • nowrap : 줄바꿈 X
  • pre : 노트패드에 입력한 enter, tab, 띄어쓰기 반영

text가 box보다 클 때 말줄임표 나오게 하기

  • white-space:nowrap : 자동 줄바꿈 안되게하기
  • overflow:hidden : 박스를 넘은 컨텐츠 안보이게 하기
  • text-overflow:ellipsis : 말줄임표 생성하기

text-transform : 소문자 or 대문자로

text-transform : uppercase 대문자로 lowercase 소문자로

  • uppercase : strength > strength
  • lowercase : STRENGTH > strength

text-decoration : line 관련

none 줄없음 line-through 취소선 overline 밑줄을 글 위로 color 밑줄 색상 지정

  • none : 배떡로제떡볶이 > 배떡로제떡볶이
  • line-through : 배떡로제떡볶이 > 배떡로제떡볶이
  • line-overline : 배떡로제떡볶이 > 배떡로제떡볶이
  • text-decoration-color : 배떡로제떡볶이 > 배떡로제떡볶이

text-indent : 들여쓰기

  • 검색엔진 최적화 위해 사용 text-indent:-9999999px;
  • text-indent 10% 적용
