블럭 요소와 인라인 요소를 각각의 모습으로 변신시켜주는 요소

  • inline : block 요소를 inline 요소로 변환 예시 : list 가로 배치
  • block : inline 요소를 block 요소로 변환 예시 : img 1열 배치
  • inline-block : 겉인속블 예시 : 나란히 하되 패딩값 갖고 싶을 때
  • none : html을 통해 적힌 텍스트를 화면에 보이지 않도록 숨김
  • - h1 등 검색을 위해 텍스트를 작성하지만 이미지로 대체할 경우 텍스트 숨김 필요
  • - 숨김이 가능한 다른 요소 : 1) visibility : visible, hidden 2) opacity: 1, 0
  • table : table로 변환
  • - 특징 : width 값 없이 중앙정렬 가능
  • - table-cell : td가 아닌 요소를 td처럼 변환
  • - vertical-align : 수직 중앙정렬 table에서만 적용
