선택자 1 선택자 2 바로가기

tag 선택자

: 태그를 선택하여 스타일 지정

class 선택자

: class 이름을 부여하여 스타일 지정

  • 주의 : 한 문서에 다중 사용 가능
  • 방법 : class = " name "
  • style에서 class 호명 : . name

id 선택자

: id 이름을 부여하여 스타일 지정

  • 주의 : 한 문서에 한 번만 사용 가능
  • 방법 : id = " name "
  • style에서 id 호명 : # name

class, id 이름 선정

  • 완전한 단어 / 띄어쓰기 X / 영어로 시작 / 숫자 & 일부 특수기호(- , _ , $) 혼용 O
  • 복합단어 : red-box 케밥케이스 red_box 스네이크케이스 redBox 카멜케이스

속성 선택자

: 태그의 속성을 선택하여 해당 속성을 가진 태그만 스타일 지정

  • 방법 1 : tag [속성] {style}
  • 방법 2 : tag [속성 = " 속성값 " ] {style}
  • 방법 3 : tag [속성 ^ = " 속성값 " ] {style} a로 "시작하는" 모든 속성 선택
  • 방법 4 : tag [속성 $ = " 속성값 " ] {style} a로 "끝나는" 모든 속성 선택
  • 방법 5 : tag [속성 * = " 속성값 " ] {style} a를 "포함하는" 모든 속성 선택

자식 / 자손 선택자

: 부모 바로 아래에 있는 자식/자손을 선택하여 스타일 지정

  • 방법 : 부모 > 자식 {style}
  • 방법 : 조상 띄어쓰기 자손 {style}

일반 형제 선택자 : ~

: 인근에 위치한 형제를 모두 선택하여 스타일 지정

  • 방법 : 형제1 ~ 형제2 {style}
  • 예시 : p ~ div {style} p 아래에 위치한 div 모두 선택

인접 형제 선택자 : +

: 같은 부모를 둔 인근에 위치한 특정 형제만 선택하여 스타일 지정

  • 방법 : 형제1 + 형제2 {style}
  • 예시1 : p + div {style} p 바로 아래에 위치한 div만 선택
  • 예시2 : li + li + li {style} 3번째 li부터 선택
  • li (기준) + li (여기부터 적용) {style}
