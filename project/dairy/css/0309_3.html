media query

정의

화면 크기에 따른 각각의 속성 값을 지정하여, 여러 가지 화면을 구현하는 기술

사용방법

@media 범위 only, all, 생략 대상 screen, print, 생략 and (조건문) {실행문}

  • 범위 = only, all, 생략
  • 대상 = screen, print, 생략
    • screen : 컴퓨터 스크린, 휴대폰 액정
    • print : 인쇄 장치
  • (조건문) : 해당 크기 설정 가능
  • {실행문} : 조건에 따른 실행을 설정

예시

  • breakpoint range 더보기
  • 너비 0 - 1440px pc
  • @media screen and (max-width:1440px) {body{background:red;}
  • 너비 0 - 1280px pc
  • @media screen and (max-width:1280px) {body{background:red;}
  • 너비 0 - 1024px pc
  • @media screen and (max-width:1024px) {body{background:blue;}
  • 너비 0 - 720px 태블릿
  • @media screen and (max-width:720px) {body{background:green;}
  • 너비 0 - 360px 모바일
  • @media screen and (max-width:360px) {body{background:pink;}

너비별 디자인 별도 설정

  1. link rel="stylesheet" href="layout1.css" 800px 이상 = layout1.css 적용
    media="only screen and (min-width:800px)"
  2. link rel="stylesheet" href="layout2.css" 321px 이상 799px 이하 = layout2.css 적용
    media="only screen and (min-width:321px) and (max-width:799px)"
  3. link rel="stylesheet" href="layout3.css" 320px 이하 = layout3.css 적용
    media="only screen and (max-width:320px)"

인쇄용 디자인 별도 설정

  • 본문 css 와 인쇄용 css 2개 link 걸기
    1) link rel="stylesheet" href="layout.css" media="screen"/
    2) link rel="stylesheet" href="print.css" media="print"/

주의사항

  • 띄어쓰기 유의
  • 숫자 겹치지 않게 쓰기
  • {실행문} 중괄호 2번 반복 유의
  • 순서(오름차순, 내림차순) 지키기
  • 크롬 미반영시, 더보기 - 설정 - 인터넷 사용기록 삭제 (비밀번호, 사이트 설정 제외)

관련 태그

  • meta name="viewport" content="width=device-width, initial-scale=1.0"
  • - width = device-width : 너비는 장치의 너비와 동일하게
  • - initial-scale = 1.0 : 접속 초기화면 배율을 1로
  • min-width / height : 최소 너비 / 높이
  • max-width / height : 최대 너비 / 높이
