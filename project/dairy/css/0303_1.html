  • - 50%, 50% : 중앙

  • scale : 크기 2 / 0.5 등등
  • perspective : 원근감 100px
  • translate : 이동 100px
  • - Z : 깊이

transition : 변화

- 변화 전후의 중간단계 생성하여 애니메이션처럼 보이게 함
- transition은 가상선택자가 아닌 변화가 일어나기 전의 태그에 설정함!
-> 가상선택자에 transition 설정 : 마우스 내리면 스르륵 X 뚝딱 O

  • property : 변화될 속성 width, transform
  • duration : 변화가 지속될 시간 2s
  • timing-function: : 변화의 속도 ease, ease-in, ease-out, linear 등

transition 적용 범위

  • Box model : width height max-width max-height min-width min-height
  • padding margin border-color border-width border-spacing
  • Background : background-color background-position
  • 텍스트 : color font-size font-weight letter-spacing line-height text-indent
  • text-shadow vertical-align word-spacing
  • 좌표 : top left right bottom
  • 기타 : opacity outline-color outline-offset outline-width visibility z-index

요소를 보였다, 안보였다 하는 3총사

1. display
- block 보이기 none 안보이기
- none : 처음부터 없던 것처럼 흔적(자리)조차 남지 않고 사라짐
2. visibility
- visible 보이기 hidden 안보이기
- hidden : 자리가 흔적으로 남고, 박스 크기에 영향을 미침
3. opacity
- 1 보이기 0 안보이기
- 0 : 자리가 흔적으로 남고, 박스 크기에 영향을 미침
