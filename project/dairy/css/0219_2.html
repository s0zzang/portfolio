우선순위

- 아주 복잡하니 외부 link 방식 위주로 사용하기

1. 선언순서

나중에 선언된 스타일이 우선 적용

2. 중요도

어느 곳에 style을 지정하는지에 따라 중요도가 결정됨

우선순위 방법
1 태그 내 style 속성
2 head 내 style 태그
3 head 내 style 태그 내 @import
4 link 태그로 연결된 css 파일
5 link 태그로 연결된 css 파일 내 @import
6 브라우저 디폴트 스타일시트

3. 명시도

작성 순서에 상관없이 점수에 따라 우선순위가 결정됨

점수 지정방법
최우선 !important
1000 인라인스타일
100 ID
10 Class 속성선택자 가상클래스
1 태그선택자
0 전체선택자 (*)
