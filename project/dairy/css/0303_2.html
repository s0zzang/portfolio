overflow, clip

overflow

박스와 컨텐츠의 크기와 관계된 스타일

  • hidden : 박스에서 넘친 컨텐츠 숨김
  • scroll : 박스에서 컨텐츠가 넘치든 안넘치든 스크롤바 생성
  • - overflow-y:scroll : y축 스크롤 무조건 생성
  • auto : 박스에서 컨텐츠가 넘치면 스크롤바 O, 안넘치면 X
  • 예시

hidden 박스보다 컨텐츠가 길어지면 짤리지

scroll 박스보다 크든 작든 스크롤이 생기지

auto 알아서 생기지

clip

포토샵 클리핑마스크 처럼 원하는 부분만 보여지게 하기

  • rect ( top, right, botton, left )
  • position : absolute / fixed 일때만 사용 가능

짜장이 코만 보이게 하고 싶은데 clip 적용이 안된다
