position

- 절대 / 상대 위치 설정
1) position 설정값 설정
2) 좌표 top, right, left, bottom 로 이동하기

  • position : absolute
    • 절대적 위치 설정
    • 위치 시작점 : 브라우저 상단 좌측
    • 부모 (position:relative) 자식(position:absolute) 경우
    • : 부모 요소 좌측 상단 시작
  • position : relative
    • 상대적 위치 설정
    • 위치 시작 점 : 나 / 부모 요소의 기본 위치
  • position : fixed
    • 고정 위치 설정
    • 위치 시작 점 : 브라우저 상단 좌측
    • 특징 : 스크를해도 위치가 고정됨

z-index

  • position 속성 설정 후 원하는 깊이에 위치하기 위해 사용
  • 사용하지 않았을 땐 z-index = 1 / +, - 가능
  • z-index : 5 = 지상 5층
  • z-index : -1 = 지하 1층
  • 예시 : div1 > div2 z-index:2 + div3 z-index:3
  • 형제인 div2와 div3의 깊이 차이를 주기 위해 사용함

부모 position에 따른 absolute 자식 시작점

  • absolute 부모 > absolute 자식 : 부모요소 좌측 상단
  • relative 부모 > absolute 자식 : 부모요소 좌측 상단
  • fixed 부모 > absolute 자식 : 부모요소 좌측 상단 하지만, 부모 다음 요소들이 치고 올라가 보이지 않음 !!
  • X 부모 > absolute 자식 : 브라우저 좌측 상단

position을 사용한 수평, 수직 중앙정렬

  • 부모 (300px X 300px), 자식 (100px X 100px)
