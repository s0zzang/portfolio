layout

  • .container
  • .row : 12개의 열(col)까지 가능
  • .col : col-n n = 너비, 12개가 넘어가면 줄바꿈됨

ver 3

  • col-xs-n : 768px 이하 device일때, col의 너비 = n
  • col-sm-n : 768px 이상 device일때, col의 너비 = n
  • col-md-n : 992px 이상 device일때, col의 너비 = n
  • col-lg-n : 1200px 이상 device일때, col의 너비 = n
  • col의 높이가 자동으로 설정되지 않음
  • width

ver 4

  • col-lg-n / col-n : large device일때, col의 너비 = n
  • col의 높이가 자동으로 설정
  • max-width
