nesting

- 포함관계를 적용한 작성법
- & : 나 자신/문자열 그 자체

  • 부모 { ~, ~, 자식 { ~, ~ } }
  • 부모 { ~, ~ }
  • 부모 자식 { ~, ~ } 자손선택자 적용
  • 부모 { ~, ~, &>자식 { ~, ~ } }
  • 부모 > 자식 { ~, ~ } 자식선택자 적용
  • 부모 : #box1
  • 자식 : #box1-title
  • #box1-title = &-title
  • &, &-title { ~, ~ }

미디어쿼리

  • div { ~, ~, @ media screen (max-width:500px) { ~, ~ } }

mixin

- 다른 속성을 하나의 코드 블록으로 정의해서 재사용
- @mixin "name" ($인자) {속성}

  • mixin X
  • box1 { background : red; font-size : 14px; }
  • box2 { background : blue; font-size : 16px; }
  • box3 { background : gray; font-size : 20px; }
  • mixin O
  • @mixin fontSizeBgColor( $fontSize, $bgColor )
    { font-size : $fontSize; background-color : $bgColor; }
  • box1 { @include fontSizeBgColor ( 14px, red ); }
  • box2 { @include fontSizeBgColor ( 16px, blue ); }
  • box3 { @include fontSizeBgColor ( 20px, gray ); }
  • mixin defalut 값 할당
  • @mixin fontSizeBgColor( $fontSize:14px, $bgColor )
    { font-size : $fontSize; background-color : $bgColor; }
  • box1 { @include fontSizeBgColor ( red ); }
  • box2 { @include fontSizeBgColor ( 16px, red ); }
  • 안쓰면 기본 값 할당
